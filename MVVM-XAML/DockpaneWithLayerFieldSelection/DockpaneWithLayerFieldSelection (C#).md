## How to use the sample
<!-- TODO: Explain how this sample can be used. To use images in this section, create the image file in your sample project's screenshots folder. Use relative url to link to this image using this syntax: ![My sample Image](FacePage/SampleImage.png) -->
1. In Visual Studio build the solution and run the debugger to open ArcGIS Pro.
2. In ArcGIS Pro open any project with a map containing feature layers (the sample only works for feature layers).    
3. Select the 'Add-in' tab, and then click on 'Show Layers & Field Selection' to open the test dockpane.  
4. Select a Feature Layer from the dropdown and note that the Field dropdown is populated with the field names for that feature layer.  
![Screen1](Screenshots/Screen1.png)  
5. Select a a Field Name from the dropdown to populated the 'Selection'.  
![Screen3](Screenshots/Screen2.png)
